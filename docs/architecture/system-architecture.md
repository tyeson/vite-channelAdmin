# 系统架构

## 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                         用户浏览器                           │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      Nginx (反向代理)                        │
│                    http://localhost:80                      │
└─────────────────────────────────────────────────────────────┘
                    │                    │
                    ▼                    ▼
┌─────────────────────────┐  ┌─────────────────────────────────┐
│   前端静态资源           │  │        后端 API 服务             │
│   Vue3 + Vite           │  │    Spring Boot 3.2              │
│   http://localhost:3000 │  │    http://localhost:8080        │
└─────────────────────────┘  └─────────────────────────────────┘
                                        │
                    ┌───────────────────┼───────────────────┐
                    ▼                   ▼                   ▼
          ┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
          │     MySQL       │  │     Redis       │  │   文件存储      │
          │   数据持久化     │  │   缓存/会话     │  │   本地/OSS      │
          └─────────────────┘  └─────────────────┘  └─────────────────┘
```

## 前端架构

```
frontend/
├── src/
│   ├── api/              # API 接口管理
│   ├── components/       # 公共组件
│   │   ├── common/       # 通用组件
│   │   └── business/     # 业务组件
│   ├── composables/      # 组合函数
│   ├── directives/       # 自定义指令
│   ├── layouts/          # 布局组件
│   ├── locales/          # 国际化
│   ├── pages/            # 页面组件
│   ├── router/           # 路由配置
│   ├── stores/           # 状态管理
│   ├── styles/           # 样式文件
│   ├── types/            # 类型定义
│   └── utils/            # 工具函数
```

## 后端架构

```
backend/
├── src/main/java/com/channel/admin/
│   ├── config/           # 配置类
│   ├── common/           # 公共模块
│   │   ├── constant/     # 常量
│   │   ├── enums/        # 枚举
│   │   ├── exception/    # 异常
│   │   ├── result/       # 响应
│   │   └── utils/        # 工具
│   ├── security/         # 安全认证
│   └── modules/          # 业务模块
│       ├── auth/         # 认证模块
│       └── system/       # 系统模块
```

## 认证流程

1. 用户登录 → 后端验证 → 生成 JWT Token
2. 前端存储 Token → 请求携带 Token
3. 后端过滤器验证 Token → 解析用户信息 → 放行请求
